Sieve — Popup & Click Hijack Blocker test harness (STRICT)

Checking environment…

How to use: serve this folder over http://localhost (python -m http.server), load the unpacked extension, open this page. Watch the console for [Sieve] Blocked… lines and the popup's count.

Strict mode blocks ALL script-driven new windows by default and shows an on-page Allow prompt. So the B-tests must BLOCK (a prompt appears) and the A-tests must stay ALLOWED (no prompt — these are normal links, downloads, etc.). Run with the module OFF first (B-tests will FAIL = dormant, A-tests PASS), then ON (everything PASS). Each block pops the real Allow prompt; this harness dismisses it automatically so tests can run back-to-back.

5 · Recovery (manual)

When a popup is blocked you get an on-page prompt: Allow (opens it once), Always allow this site (whitelists the host), or ✕. The Sieve popup also lists every blocked URL with Open anyway, and Clear list resets the count.
